---
title: "reseed inspect"
description: "Show the skills in a pack"
---

```bash
reseed inspect <pack>
```

Lists all skills in a pack and shows whether each one is present in your library.

## Example output

```
Pack "frontend":
  frontend-design (ok)
  css-reset (ok)
  accessibility (missing)
```

A skill marked `(missing)` is defined in the pack but not found in your library. You may need to create it or install it from GitHub.
