---
title: "reseed library"
description: "List all skills in your library"
---

```bash
reseed library
```

Lists all skills in your library, grouped by local and external. External skills show their source and version.

## Example output

```
Local:
  code-review
  python-typing

External:
  frontend-design  (user/repo/src/skills/frontend-design, latest)
  writing-style    (user/repo/writing-style, v1.0)
```
