---
title: "reseed status"
description: "Show skills installed in the current project"
---

```bash
reseed status
```

Displays the skills currently installed in the project's skills directory.

## Example output

```
  code-review
  frontend-design
  python-typing
```

## Global flags

- `--dir` Override the skills directory (default `.agents/skills/`)

```bash
reseed --dir .claude/skills status
```
