---
title: "reseed pack"
description: "Create or edit a pack interactively"
---

```bash
reseed pack <name>
```

Opens an interactive selector to create or edit a pack. Use arrow keys to navigate, space to toggle skills, and enter to confirm.

```
Pack "frontend" - select skills (space to toggle, enter to confirm):

> [x] accessibility
  [ ] code-review
  [x] css-reset
  [x] frontend-design
  [ ] python-typing

q/esc to cancel
```

If the pack already exists, its current skills are pre-selected.

Selecting no skills and pressing enter removes the pack.
