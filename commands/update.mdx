---
title: "reseed update"
description: "Fetch latest versions of external skills"
---

```bash
reseed update
```

Re-fetches external skills from GitHub into your library. Skills set to `latest` in `reseed.yaml` get the newest version. Skills pinned to a specific tag are skipped.

## Example output

```
  ~ frontend-design...
  - sql-safety (pinned at v1.0, skipped)
```
